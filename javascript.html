LA DECLARATION DES VARIABLES, OPERATEURS ET TYPES DE DONNEES

LA DECLARATION DE VARAIABLES

En Javascript la déclaration des variables se fait à l'aide du mot clé let avec une version plus ancienne qui est var.
Pas besoin de donner préciser le type de la variable comme en c++ il sufit juste de l'initialiser et le language lui attribuera le type de la valeur que vous lui avez attribué.Le nom de la variable doit vérifier la convention camel case : en gros c'est de ne jamais commencer le nom d'une varaiable avec une majuscule, les noms de variables doivent être le plus explicite possible et quand on as plusieurs mots qui forment la variable on les collent en les distinguant par la majuscule au niveau de chaque nouveaux mots sauf pour le premier.
Comme en Python le typage en javascript est assez permissif c'est à dire qu'une variable à laquelle on avait affecté une valeur de type boléen peut devenir de type entier si on lui affecte une valeur de cet type.
L'initialisation parlons-en c'est a peu près la même chose que dans tout les autres languages à la différence près que ici elle est obligatoire dès la création de la variable . Mieux pour créer une variable il faut l'nitialiser.
Pour le faire on utilse le code suivant 
                    
    let numberOfChoice = 10;

LES OPERATEURS Javascript

Comme dans tous les languages JS aussi possède les opérateurs de base ou OPERATEURS ARITHMETIQUES comme : +,-,*,/,%,** et leurs significations !
Nous avons aussi des opérateurs affiliés à chacun de ses operateurs de base qui sont appelésOPERATEURS D'AFFECTATION telsques : -=,/=,*=,+=,%=,= .
Pour exemple on a : let numberOfChoice = 10; numberOfChoice-=10; //numberOfChcede vient egale 0 /** la traduction de l'operateur donne quelquechose comme : numberOfChoice = numberOfChoice - 10 et la même chose vaut pour les operateurs de cet type */

LES OPERATEURS D'INCREMENTATION ET DE DECREMENTATION

Il s'agit de ++ et --
De manière simple quand je lance par exemple 
                    
    let numberOfChoice = 2;
    numberOfChoice++
    // numberOfChoice passe de 2 à 3
    // et inversement pour la décrémentation
